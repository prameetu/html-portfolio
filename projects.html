My Projects

  • Blind Vision - AI
    • An model for assisting blind people with captioning image via a web app
    • Model uses two neural networks, a CNN based image feature extractor and an LSTM based sentence generator.
    • The back-end is designed with Flask which allows users to feed image to model and sentences describing image are read aloud to user.
    • Major Tools : Python,Numpy,Keras, Pillow

  • ROCK-PAPER-SCISSOR-AI
    • An OpenCV application to play Rock Paper Scissors game using your hand gestures VS Computer.
    • Model uses pretrained VGG16 neural networks a CNN based image feature extractor.
    • Model is trained on self-made image dataset generated by python script using OpenCV that captures hand gestures.
    • Major Tools : Python,Tensorflow,Keras, OpenCV
